# تصحيح سعر حجز يدوي (قبل ما الجلسة تكمّل)

**الطلب:** 25913 · **الجلسة:** 846455 · **HPOS:** شغال

> اعمل نسخة احتياطية لقاعدة البيانات قبل ما تبدأ.

**استبدل `السعر_الصح`** في كل الخطوات بالرقم الصح (مثال: `350`).

---

## قبل التعديل

- [ ] افتح phpMyAdmin واختار قاعدة بيانات الموقع
- [ ] بادئة الجداول عندكم: **`wpds_`** (الاستعلامات جاهزة بيها)

### 1) الجلسة لسه مفتوحة؟

```sql
SELECT session_status, order_id
FROM wpds_snks_provider_timetable
WHERE ID = 846455;
```

- [ ] `order_id` = **25913**
- [ ] `session_status` = **open** (مش completed)  
  → لو completed: **وقف** وكلم الدعم الفني

### 2) مفيش أرباح اتسجّلت؟

```sql
SELECT id FROM wpds_snks_booking_transactions
WHERE ai_session_id = 846455 OR ai_order_id = 25913;
```

- [ ] النتيجة **فاضية** (مفيش صفوف)  
  → لو فيه صفوف: **وقف** وكلم الدعم الفني

## التعديل

### 3) جيب رقم سطر المنتج في الطلب

```sql
SELECT order_item_id
FROM wpds_woocommerce_order_items
WHERE order_id = 25913 AND order_item_type = 'line_item';
```

- [ ] اكتب رقم `order_item_id` هنا: _______________

### 4) عدّل إجمالي الطلب

```sql
UPDATE wpds_wc_orders_meta SET meta_value = 'السعر_الصح'
WHERE order_id = 25913 AND meta_key = '_main_price';

UPDATE wpds_wc_orders_meta SET meta_value = 'السعر_الصح'
WHERE order_id = 25913 AND meta_key = 'ai_total_amount';

UPDATE wpds_wc_orders SET total_amount = السعر_الصح
WHERE id = 25913;
```

- [ ] نفّذت الاستعلامات الثلاثة

### 5) عدّل سعر سطر الجلسة

(استبدل `رقم_السطر` بالرقم من الخطوة 3)

```sql
UPDATE wpds_woocommerce_order_itemmeta SET meta_value = 'السعر_الصح'
WHERE order_item_id = رقم_السطر
AND meta_key IN ('_line_total', '_line_subtotal', 'total', 'subtotal');
```

- [ ] نفّذت الاستعلام
