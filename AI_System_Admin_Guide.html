üöÄ 1. System Overview

What is Jalsah AI System?

The Jalsah AI System is a comprehensive healthcare management platform that integrates artificial intelligence to streamline patient-therapist interactions, appointment scheduling, and medical consultations.

üéØ Key Features

  • AI-Powered Patient Matching: Automatically matches patients with suitable therapists based on diagnosis
  • Smart Appointment Scheduling: Intelligent booking system with real-time availability
  • Integrated Communication: WhatsApp and email automation for patient engagement
  • Advanced Analytics: Comprehensive reporting and insights
  • Rochtah Integration: Specialized consultation platform for prescriptions

üîß System Requirements

  • WordPress 5.0 or higher
  • PHP 7.4 or higher
  • MySQL 5.7 or higher
  • WooCommerce plugin (for payment processing)

    ü§ñ 8. ChatGPT Integration

    Purpose

    Integrate OpenAI's ChatGPT to provide intelligent diagnosis recommendations and enhance patient-therapist matching.

    üîß API Configuration

  • OpenAI API Key: Your unique API key from OpenAI
  • Model Selection: GPT-3.5 Turbo, GPT-4, or GPT-4 Turbo
  • System Prompt: Instructions for the AI behavior

  • Temperature: Creativity level (0.0-2.0)

    • üìù System Prompt Example

    üß™ Testing Integration

    Example Test:

    Expected Response: "Based on your symptoms, I recommend 'Anxiety Disorders' as the most suitable diagnosis. Your symptoms of persistent anxiety, sleep disturbances, and panic attacks are consistent with Generalized Anxiety Disorder (GAD)."

    üìä Available Diagnoses

    The system automatically provides ChatGPT with the current list of available diagnoses from your database, ensuring recommendations are always relevant to your services.

    ‚ö†Ô∏è Important Notes:

    • Keep your API key secure and never share it
    • Monitor API usage to control costs
    • Test the integration regularly
    • ChatGPT recommendations are suggestions, not medical diagnoses

    üîß 13. Troubleshooting

    Common Issues & Solutions

    Quick fixes for the most common problems you might encounter.

    ‚ùå Database Table Issues

    Problem: "Table doesn't exist" errors

    Solution:

    1. Go to the specific admin page (e.g., Rochtah Integration)
    2. Look for "Database Status" section
    3. Click "Create Table Manually" if table is missing
    4. Refresh the page and try again

    ü§ñ ChatGPT Integration Issues

    Problem: ChatGPT not responding

    Solution:

    1. Check your OpenAI API key is correct
    2. Verify you have sufficient API credits
    3. Test the integration with a simple prompt
    4. Check your internet connection

    üì± WhatsApp Issues

    Problem: WhatsApp messages not sending

    Solution:

    1. Verify your WhatsApp Business API credentials
    2. Check message templates are approved
    3. Ensure phone numbers are in correct format
    4. Test with a single message first

    üìß Email Issues

    Problem: Emails not being sent

    Solution:

    1. Check WordPress email configuration
    2. Verify SMTP settings if using custom email
    3. Test email functionality
    4. Check spam/junk folders

    üìä Performance Issues

    Problem: Slow loading or timeouts

    Solution:

    1. Check server resources and memory
    2. Optimize database queries
    3. Clear WordPress cache
    4. Update to latest plugin version
